# 股票分红计算器

一个用于计算股票分红回本周期的 Web 应用，支持红利再投入和不再投入两种模式的对比分析。

## 功能特性

- 📊 **分红回本计算** - 计算股票分红多少年能够覆盖买入成本
- 🔄 **双模式对比** - 支持红利再投入和不再投入两种策略
- 📈 **送股转增支持** - 自动计算送股、转增股对持股数量的影响
- 💰 **再投入限制** - 红利再投入按 A 股规则（100股整数倍）买入
- 📉 **数据可视化** - 图表展示累计分红对比
- 🔍 **股票搜索** - 支持股票代码和名称搜索

## 技术栈

### 前端
- Vue 3
- Chart.js
- Axios
- Vite

### 后端
- Python 3
- Flask
- 数据源支持：
  - [akshare](https://github.com/akfamily/akshare) - 默认数据源
  - [baostock](http://baostock.com/) - 备用数据源（更稳定）

## 快速开始

### 1. 后端启动

```bash
# 进入后端目录
cd backend

# 创建虚拟环境（首次运行）
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 安装依赖（首次运行）
pip install -r requirements.txt

# 启动后端服务
# 使用 akshare 数据源
python app.py

# 或使用 baostock 数据源（推荐）
python app2.py
```

后端服务运行在 http://127.0.0.1:5001

### 2. 前端启动

```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

前端服务运行在 http://127.0.0.1:3000

### 3. 访问应用

浏览器打开 http://127.0.0.1:3000

## 使用说明

1. **搜索股票** - 输入股票代码（如 600519）或名称（如 贵州茅台）
2. **设置参数**
   - 买入年份：股票买入的年份
   - 买入价格：可选，留空则自动获取当年价格
   - 买入股数：购买的股票数量
3. **开始计算** - 点击按钮查看结果

## 计算结果说明

### 不再投入
- 现金分红不用于再购买股票
- 送股、转增股会增加持股数量
- 累计分红 = 所有年度现金分红之和

### 再投入
- 现金分红用于购买更多股票
- 买入规则：100股的整数倍
- 不足100股的现金保留累计
- 持股数量会随时间增加

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/search_stock?keyword=xxx` | GET | 搜索股票 |
| `/api/calculate` | POST | 计算分红回本 |
| `/api/stock_info/<code>` | GET | 获取股票信息 |

### 计算接口参数

```json
{
  "stock_code": "600519",
  "buy_year": 2015,
  "buy_price": 200,      // 可选
  "buy_shares": 1000     // 默认 1000
}
```

| 指标 | 不再投入 | 再投入 |

| 回本年限 | 11年 | 9年 |
